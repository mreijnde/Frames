Demonstration of multi-dimensional DataFrame functionality

Multi-dimensional indexing is possible in DataFrame class by using the MultiIndex class type for rows and columns. In the design the syntax of the DataFrames with a MultiIndex is chosen as much as possible close to the behavior of a DataFrame with only a single dimension index using the Index class.
